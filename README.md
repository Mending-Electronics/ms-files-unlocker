# MS Unlocker App

## Overview

The **MS Unlocker App** simplifies Excel security management for files ranging from 2007 to 2016 versions. This robust platform empowers you to:
- **Reset or remove passwords** and protections.
- **Decode common VBA obfuscations**.
- **Analyze for potential malware threats** in VBA scripts.

## Features

- **Password Management**: Easily reset or remove passwords and protections from Excel files.
- **VBA Decoding**: Decode commonly used VBA obfuscations to understand the underlying code.
- **Malware Analysis**: Identify potential malware threats hidden in VBA scripts, ensuring the safety of your files.

## Compatibility

- Supports Excel files from 2007 to 2016 versions.

## How to Use

1. **Upload your Excel file**: Simply drag and drop your file into the application.
2. **Choose an action**: Select whether you want to reset/remove passwords, decode VBA, or analyze for malware.
3. **Get results**: The app quickly processes your request and provides you with the necessary output.

To run MS Unlocker App locally, follow these steps:

- Install python 3.11 or higher
- Create a python venv
- Clone the repository
- Install necessary packages


```bash
python -m venv venv

cd venv

git clone "https://github.com/Mending-Electronics/ms-files-unlocker.git"

Scripts\activate

pip install -r requirements.txt
```
